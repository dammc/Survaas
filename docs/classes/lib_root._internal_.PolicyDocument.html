_splitDocument

  • _splitDocument(
        scope: IConstruct,
        selfMaximumSize: number,
        splitMaximumSize: number,
    ): Map<PolicyDocument, PolicyStatement[]>
  • Internal

    Split the statements of the PolicyDocument into multiple groups, limited by their size

    We do a round of size-limited merging first (making sure to not produce statements too large to fit into standalone policies), so that we can most accurately estimate total policy size. Another final round of minimization will be done just before rendering to end up with minimal policies that look nice to humans.

    Return a map of the final set of policy documents, mapped to the ORIGINAL (pre-merge) PolicyStatements that ended up in the given PolicyDocument.

    Parameters

    • scope: IConstruct
    • selfMaximumSize: number
    • splitMaximumSize: number

    Returns Map<PolicyDocument, PolicyStatement[]>
