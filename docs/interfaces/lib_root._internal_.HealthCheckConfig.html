Interface HealthCheckConfig

Settings for an optional Amazon Route 53 health check. If you specify settings for a health check, AWS Cloud Map associates the health check with all the records that you specify in DnsConfig. Only valid with a PublicDnsNamespace.

interface HealthCheckConfig {

    resourcePath?: string;
